Changelog

v0.2.1

- Added scripts to desktop edition, now installing the dependencies, running the game and compiling the game is easier.

v0.2

- Added localization:

	+ new Global Variables:

		* currentLanguage: defaults to "default",current game language.

		* availableLanguages: defaults to empty, array with all the available localizations.

	+ new fields:

		* localization: Array with objects that contain a field named "language" with the localization id and "text" with the localized text, usable in choice and text.

		* language: Sets the language to the value of the field, usable in redirect type choices.

	+ new value for "type" choice field:

		* language: Choice button that changes the language of the game.


		
- Refactored sound API:

	+ Now the sound system is an object, soon more parts of the engine will be changed to objects, the final objective is to not longer have global variables.



- Now buttons get disabled when pressed to avoid click spamming.

Scripts 101

October engine games are written in little pieces called "scripts".
This scripts are JSON documents with information about a game state.
The engine uses that information to show the player information and process its responses.

Scripts are stored in the "script" folder inside the root folder of the game.
They use the .json extension and the rest of the filename without the extension is the identifier of that script and can be use to call it from other scripts.
If you set the global variable "key" to a value you must provide the scripts files with the extension .enc and encrypted using AES and the value of "key" as passphrase.

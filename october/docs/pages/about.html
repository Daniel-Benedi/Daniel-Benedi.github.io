What is October Engine?

October engine is a game engine designed to facilitate and accelerate development of visual novels.
Games are written in little pieces called "scripts" and can run in the browser and desktop operating systems (Windows,MacOS and Linux).

Features

You can:

  • Show text dialogs (Duh).
  • Use Background images.
  • Put a sprite or image to show something or someone relevant to the conversation or text.
  • Present the player with different options that can change the outcome of the game.
  • Ask the player for input that you can use later to make choices or insert into the story script.
  • Create sprites and place them into an interactive scene
  • And much more
